# 🐾 Pixie

Pixie is an AI-powered web application designed to help communities protect and care for cats by connecting pet owners, volunteers, and animal lovers through a simple and modern platform.

Whether a cat is lost, found wandering the streets, or in need of urgent care, Pixie makes it easier to report, search, and identify cats using Computer Vision and AI.

> 🚧 **Project Status:** Work in Progress

---

# ✨ Overview

Pixie provides a centralized platform where users can:

* 🐱 Report lost cats.
* 🥣 Report stray or injured cats that need food, rescue, or medical care.
* 🔍 Search existing reports.
* 🤖 Use an AI-powered **Find a Cat** scanner to identify a cat's breed and check whether it matches any existing lost or stray cat reports.
* 👤 Manage reports through a personal profile.
* 🔔 Receive notifications when potential AI matches are found.

The goal is to increase the chances of reuniting lost cats with their owners while helping stray cats receive the care they need.
